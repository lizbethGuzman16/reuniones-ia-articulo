# Licencias y fuentes

## Datos

El dataset usado es ICSI MRDA distribuido mediante SILICONE. La ficha pública del benchmark declara licencia **CC BY-NC-SA 4.0**. Los datos se incluyen para reproducción académica no comercial y con atribución.

- Corpus original: Shriberg et al. (2004), *The ICSI Meeting Recorder Dialog Act (MRDA) Corpus*.
- Benchmark: Chapuis et al. (2020), *Hierarchical Pre-training for Sequence Labelling in Spoken Dialog*.
- Repositorio: https://github.com/eusip/SILICONE-benchmark

Las URLs y hashes exactos se encuentran en `data/raw/dataset_manifest.json`.

## Código y dependencias

El proyecto integra código desarrollado para esta entrega con el sistema base proporcionado al equipo. Las dependencias conservan sus propias licencias. Antes de redistribuir comercialmente, revise especialmente la licencia no comercial de los datos.
